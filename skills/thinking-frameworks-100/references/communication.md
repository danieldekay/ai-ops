# Communication & Argumentation

Frameworks for structuring arguments, evaluating claims, and communicating ideas persuasively.

| #   | Framework                                   | Description                                                                                                                                                        | Origin                        |
| --- | ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------- |
| 97  | **Steel Manning**                           | Represent an opponent's argument in its strongest possible form before attempting to rebut it. The opposite of straw-manning. Leads to more productive discourse.  | Rationalist community         |
| 98  | **Straw Manning**                           | (Anti-pattern) Weakening an opponent's argument to make it easier to attack. Recognizing this fallacy helps you avoid it and call it out in others.                | Logic/rhetoric                |
| 99  | **Toulmin Model of Argumentation**          | Structure arguments with: Claim, Data (evidence), Warrant (reasoning), Backing, Qualifier ("probably"), and Rebuttal. More nuanced than simple premise-conclusion. | Stephen Toulmin               |
| 100 | **CRAAP Test**                              | Evaluate information sources on: Currency, Relevance, Authority, Accuracy, and Purpose. A quick credibility filter for research.                                   | Sarah Blakeslee (Chico State) |
| 101 | **Minto Pyramid Principle**                 | Structure all communication answer-first, then supporting arguments, then evidence. Ensures clarity and impact in executive communication.                         | Barbara Minto                 |
| 102 | **Persuasion Triad (Ethos, Pathos, Logos)** | Persuade through credibility (ethos), emotional appeal (pathos), and logical argument (logos). All three reinforce each other.                                     | Aristotle, _Rhetoric_         |
| 103 | **Commander's Intent**                      | State the desired end-state clearly so everyone can make autonomous decisions aligned with the overall mission, even when plans break down.                        | Military doctrine             |
| 104 | **Nonviolent Communication (NVC)**          | Observation → Feeling → Need → Request. A framework for expressing needs and resolving conflicts without blame or demands.                                         | Marshall Rosenberg            |
