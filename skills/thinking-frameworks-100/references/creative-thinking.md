# Creative & Lateral Thinking

Frameworks for generating novel ideas, breaking cognitive fixedness, and expanding the solution space.

| #   | Framework                                        | Description                                                                                                                                                                | Origin                          |
| --- | ------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------- |
| 72  | **Six Thinking Hats**                            | Separate thinking into six modes (White=facts, Red=emotions, Black=risks, Yellow=benefits, Green=creativity, Blue=process) to explore a problem from all angles.           | Edward de Bono (1985)           |
| 73  | **Lateral Thinking**                             | Deliberately use non-sequential, non-logical approaches to break out of established patterns and generate unexpected solutions.                                            | Edward de Bono                  |
| 74  | **SCAMPER**                                      | Structured ideation checklist: Substitute, Combine, Adapt, Modify, Put to other use, Eliminate, Reverse. Apply each prompt to an existing product or process.              | Bob Eberle (based on Osborn)    |
| 75  | **TRIZ (Theory of Inventive Problem Solving)**   | A systematic approach that identifies contradictions and resolves them using 40 inventive principles derived from analysis of 200,000+ patents.                            | Genrich Altshuller              |
| 76  | **Mind Mapping**                                 | Start with a central concept and radiate branches of associated ideas. A visual, non-linear way to organize information and stimulate creative connections.                | Tony Buzan                      |
| 77  | **Brainstorming**                                | Generate as many ideas as possible without judgment (defer criticism, go for quantity, encourage wild ideas, combine and build).                                           | Alex Osborn (1953)              |
| 78  | **Brainwriting (6-3-5)**                         | 6 people × 3 ideas × 5 rounds. Write ideas silently, pass them around, and build on others' ideas. Avoids groupthink and anchoring.                                        | Bernd Rohrbach (1969)           |
| 79  | **Random Entry / Provocation**                   | Introduce a random, unrelated stimulus (word, image, object) to force new connections and break fixation on familiar solutions.                                            | Edward de Bono                  |
| 80  | **Morphological Analysis**                       | Define key dimensions, list possible values for each dimension, then systematically combine values to generate a full solution space — including non-obvious combinations. | Fritz Zwicky                    |
| 81  | **Reverse Brainstorming**                        | Instead of solving the problem, brainstorm ways to _cause_ the problem. Then invert each answer to find solutions.                                                         | Creative problem-solving        |
| 82  | **Analogical Thinking**                          | Transfer solutions from one domain to another. Ask "Where has a similar problem been solved before?" and adapt the solution.                                               | General creativity theory       |
| 83  | **Concept Fan**                                  | Start with the purpose, generate broad approaches, then for each approach generate multiple specific implementations. Fan out from abstract to concrete.                   | Edward de Bono                  |
| 84  | **Osborn-Parnes Creative Problem-Solving (CPS)** | Objective Finding → Fact Finding → Problem Finding → Idea Finding → Solution Finding → Acceptance Finding. The gold standard of creative problem-solving.                  | Alex Osborn, Sidney Parnes      |
| 85  | **Adjacent Possible**                            | Innovation happens at the edges of what currently exists. Map what has recently become feasible to find the ripest opportunities.                                          | Stuart Kauffman; Steven Johnson |

## Key Sources

- _Six Thinking Hats_ / _Lateral Thinking_ — Edward de Bono
- _Applied Imagination_ — Alex Osborn
- _Where Good Ideas Come From_ — Steven Johnson
- _And Suddenly the Inventor Appeared: TRIZ_ — Genrich Altshuller
